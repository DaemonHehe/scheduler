# SyncTerm

A blazingly fast, client-side university scheduler built for students. SyncTerm helps you plan your semester with zero loading screens, instant overlap detection, and local storage to keep your schedule exactly where you left it.

## Features

- **Instant Conflict Detection:** Real-time warnings for overlapping or tightly scheduled classes.
- **Auto-Generate Schedules:** Quickly generate schedules based on preferences like "Morning Classes", "Evening Classes", "Relaxed", or "Compact".
- **Local Storage:** Your courses and timetable are saved directly in your browser. No accounts or logins required.
- **Premium UI:** Built with Tailwind CSS, featuring a sleek, dark-themed interface with modern typography and fluid interactions.

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router, Turbopack)
- **Library:** [React](https://react.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **State Management:** [Zustand](https://docs.pmnd.rs/zustand/getting-started/introduction) (with LocalStorage Persistence)
- **Icons:** [Lucide React](https://lucide.dev/)

## Getting Started

First, clone the repository:

```bash
git clone https://github.com/DaemonHehe/scheduler.git
cd scheduler
```

Install the dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
